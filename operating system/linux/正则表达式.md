
```bash
grep   强大而灵活的文本搜索工具
.：匹配任意单个字符，除了换行符 \n。
^：匹配字符串的开头。
$：匹配字符串的结尾。
*：匹配前面的字符零次或多次。
+：匹配前面的字符一次或多次。
?：匹配前面的字符零次或一次。
|：或操作符，匹配两个或多个选择之一。
\：转义字符，用于取消元字符的特殊含义，例如 \. 表示匹配点字符而不是任意字符。
[]：字符集合，匹配方括号内的任意一个字符。
()：分组，用于将多个元素组合成一个单元

```

```bash
在日常工作中经常用于查找特定模式的文本内容，分析日志文件或者进行数据处理等任务。通过结合不同的选项和正则表达式，可以实现各种复杂的文本搜索和处理操作。

正则表达式（Regular Expression，简称 regex 或 regexp）
是一种用于描述字符串模式的工具。
它是由普通字符（例如字母、数字和符号）和特殊字符（称为元字符）组成的字符串，

可以用来检查字符串
是否与某种模式匹配、
查找字符串中的特定内容、
替换字符串中的文本等操作。
正则表达式在文本处理、编程语言中的字符串操作、数据验证等领域都有广泛的应用。
```

```bash
wc 命令 是一个统计行数和字符数的命令
wc -l file.txt  # 统计行数
wc -w file.txt  # 统计单词数
wc -c file.txt  # 统计字符数
```
